# Rhowyen Streamer Site
Welcome to the Rhowyen fan site — a custom site built to showcase and promote the content and social channels of Rhowyen, a streamer you support! 
This repository hosts the code behind the site using GitHub Pages.

---------

## About
This site is a personal project created and designed by Rschipman for streamer Rhowyen. It serves as a hub for Rhowyen’s online presence by linking fans to all major platforms and content spaces

---

## Purpose
The goal of this site is to provide a central place where viewers and followers can easily:
Find Rhowyen’s Twitch, YouTube, Threads & other social channels.
Connect with community links and supporter pages.
Share and grow interest in Rhowyen’s streaming content.
--

## What’s Included
The repository contains all the files needed for the website:
File	Purpose
index.html	Main webpage content
style.css	Custom styles and layout
script.js	Interactive behavior and scripts
assets/	Images and other media
.github/workflows/	GitHub Actions workflow (optional deployment steps)
--
## How It Works
This site is hosted using GitHub Pages, which turns this repository into a live website at:
https://testrhowyen.github.io/
GitHub Pages automatically serves the content from the main branch — no extra servers required..
--
